# Recommended GitHub Topics

Add these topics to your GitHub repository to improve discoverability:

## Primary Topics

| Topic | Description |
|---|---|
| `c` | Core programming language |
| `networking` | Network programming |
| `socket-programming` | BSD/POSIX socket API |
| `tcp` | TCP protocol implementation |
| `chat` | Chat application |

## Secondary Topics

| Topic | Description |
|---|---|
| `client-server` | Client-server architecture |
| `terminal` | Terminal/console application |
| `multithreading` | POSIX threads (pthreads) |
| `posix` | POSIX-compliant system programming |
| `linux` | Linux compatibility (also macOS/BSD) |

## Additional Topics

| Topic | Description |
|---|---|

| `concurrent-programming` | Concurrent client handling |

## How to Add Topics

1. Navigate to your repository on GitHub
2. Click the gear icon next to "About" on the right sidebar
3. Type each topic and press Enter
4. Click "Save"
